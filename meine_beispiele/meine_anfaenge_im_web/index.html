HTML in den Jahren vor 2000

Das Konzept zum World Wide Web (WWW) wurde von Tim Berners-Lee während seiner Arbeit am CERN entwickelt. Die erste Webseite wurde am 06.08.1991 online gestellt.

Ab ca. Mitte 1993 wurde das WWW vom CERN für die Öffentlichkeit zur Verfügung gestellt. Mit der Freigabe des Mosaic Browsers wurde das WWW dann auch für alle einfacher nutzbar.

Den Mosaic Browser habe ich ab ca. 1994 beruflich (Siemens AG) genutzt. Mangels Alternativen, musste ich den Mosaic Browser für unsere Sinix Systeme noch selbst compilieren.

Kern für die Nutzung ist die Sprache HTML. Eine Übersicht über alle HTML Versionen findet man im Wikipedia Artikel Hypertext Markup Language.

Angefangen mit eigenen Webseiten habe ich im Jahr 1995. Zu dieser Zeit war HTML 2.0 die aktuelle HTML Version. Anfang 1997 wurde die Version HTML 3.2 veröffentlicht, die ich dann länger genutzt habe. Schon Ende 1997 wurde HTML 4.0 veröffentlicht - und mit etlichen kleinen Korrekturen und Erweiterungen - durch HTML 4.01 Ende 1999 ersetzt. Diese Version war dann, zusammen mit XHTML 1.1, die Basis für HTML 5 im Jahr 2014.

Die letzte Version meine GeoCities Site (06.01.2005) wurde, wie alle anderen GeoCities Sites, archiviert und ist unter der URL https://www.oocities.org/marochess/ immer noch aufrufbar.

HTML 5

Ab dem Jahr 2000 habe ich eher weniger mit HTML gearbeitet. Die anstehende HTML 5 Version habe ich 2013/2014 zum Anlaß genommen, W3C Trainings zu den damaligen Drafts (Entwürfen) für HTML 5 zu absolvieren.

     

Nach 2014 habe ich HTML wieder weniger genutzt, da ich beruflich im Backend Bereich gearbeitet habe. Jetzt (Ende 2023) habe ich wieder mehr Zeit für Webanwendungen und es wird Zeit, sich endlich einmal tiefer mit CSS zu beschäftigen, denn bisher habe ich mich im Wesentlichen immer mit HTML und JavaScript beschäftigt und mich bei CSS auf ein Minimum beschränkt.
