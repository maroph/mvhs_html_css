LibreRing maroph: Sample Page

How to add my webring to a HTML5 page

Add the JavaScript file to the page head:
 
<head>
...
<script src="https://maroph.github.io/mvhs_html_css/librering/librering_maroph.js" defer="defer"></script>
...
</head>

Add the following div to the page body
 
<body>
...
<div id="librering_maroph"></div>
...
</body>
