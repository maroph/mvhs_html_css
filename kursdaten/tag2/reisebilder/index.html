Reisebilder

Diese Bilder habe ich während meiner Motorradreisen im Oktober 2023 aufgenommen und mit IrfanView von 4032x3024 Pixel auf 640x480 Pixel verkleinert. Für diese Seite habe ich die Dateien mit dem Tool JPEG Compression komprimiert, um Bandbreite beim Laden der Bilder zu sparen.

Zum Aufbau der Seite

Die einzelnen img Elemente habe ich in ein picture Element verpackt, damit Bilder verschiedener Größe, je nach Gerät, angezeigt werden können. Die picture Elemente habe ich mit figure und figcaption "verziert", damit jedes Bild eine dazugehörige Beschreibung enthält. 

    <figure>
        <img ...>
        <figcaption>Bildbeschreibung</figcaption>
    </figure>

Damit die Bilder eine kleine Umrandung erhalten, verwende ich im head Element den folgenden Code: 

    <style>
        img { border: 1px solid black; }
    </style>

Ein Mausklick mit der linken Taste lädt das Originalbild in einem neuen Tab. 

    <a href="..." target="_blank"<picture>...</picture></a>
